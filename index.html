Interactivity: JS

Toggle Visibility

Open Closed
aria-hidden="true"
inert 
<button onclick="openModal">
  Open Modal
</button>

<button onclick="closeModal">
  Close Modal
</button>

const openModal = () => {
  m.removeAttribute('aria-hidden')
  m.removeAttribute('inert')
}
const closeModal = () => {
  m.setAttribute('aria-hidden', 'true')
  m.setAttribute('inert', true)
}

Focus

  1. On open, focus on modal title or modal container.
  2. While open, trap focus inside modal.
  3. On close, restore focus to open trigger.

Outer Click

While open, listen for clicks outside the modal and close it.

Alternative to <button> scrim

Keyboard Close

While open, listen for Escape keyup to close.
